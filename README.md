# MT4 Example
## bar_info.mqh
ローソク足を取り扱う為のクラス  
MT4\Include\Damepo\bar_info.mqh  
にファイルを置いて使用
## currency.mqh
浮動小数点の誤差を考慮し内部的に整数値と文字列で数値管理する為のクラス  
USD/JPYを対象としている為、固定で6桁としている箇所があるので他通貨で利用する場合は注意が必要  
MT4\Include\Damepo\currency.mqh  
にファイルを置いて使用
## test.mq4
上記ライブラリのテストサンプル  
MT4\Experts\test.mq4  
にファイルを置いて使用
